# AI Resume Analyzer 📄

Upload any PDF resume and get instant AI-powered
feedback, scoring and improvement suggestions.

## Live Demo
[Click here](https://resume-analyzer-webapp-kly7uwtt8vtbanw5fwjwgn.streamlit.app)

## Features
- Upload PDF resume
- AI scores resume out of 100
- Section-by-section breakdown
- Top 5 strengths and weaknesses
- ATS compatibility score
- Missing keywords for target role
- Brutally honest one-liner verdict
- Download analysis as text file

## Tools Used
- Python, Streamlit, PyPDF2, Anthropic Claude API

## How to Run Locally
pip install streamlit PyPDF2 anthropic
Add API key to .streamlit/secrets.toml
streamlit run app.py
